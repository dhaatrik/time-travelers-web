# Time Traveler’s Web: A Portal to Past Internets

**Tagline:** Surf the Web Before It Was Cool

## Concept

Time Traveler’s Web is a fun, interactive single-page application that transports users to different eras of internet history. By selecting a year (1985, 1991, 1996, 2000, or 2005), the entire website's theme, content, and interactive elements transform to reflect the look, feel, and digital culture of that time.

Experience the evolution of the web, from the green-screen terminals of the 80s to the chaotic bling of early social media!

## Key Features

*   **Year Selector:** A dynamic dropdown allows users to choose a target year, triggering an immersive era change.
*   **Era-Specific Themes:** Each year features a unique visual theme:
    *   **1985:** Green terminal interface, ASCII art, command-line style, CRT screen effects, and a simulated boot-up sequence.
    *   **1991:** Mosaic-style early HTML, table layouts, basic colors, classic blue links, and "Best Viewed With..." badges.
    *   **1996:** GeoCities vibe with sparkly text, animated GIFs, marquees, tiled backgrounds, hit counters, and a guestbook.
    *   **2000:** Flash-like intros (simulated), early e-commerce design aesthetics, popups galore, and skeuomorphic button touches.
    *   **2005:** MySpace aesthetics, early social media "profile" sections, embedded media player (simulated), custom cursors, and glitter text.
*   **Dynamic & Expanded Content:**
    *   **Dynamic News Feed:** News headlines are dynamically generated by the Gemini API, reflecting the tech culture and events of the selected year.
    *   **Retro Ads & Popups:** Era-appropriate advertisements, with text dynamically generated by the Gemini API for a unique experience on each visit. Some ads appear as blinking text or classic intrusive popups.
    *   **Marquee Text:** Scrolling text banners, especially prominent in the 1996 era.
    *   **Memes & Culture Section:** Displays iconic memes (as placeholder images/captions) and notable cultural references from each period.
*   **Mini Tools & Interactive Elements:**
    *   **Guestbook:** Sign a retro guestbook (entries saved per era in local storage).
    *   **HTML Tutorial:** A basic HTML tag guide (1991).
    *   **Flash Animation Demos (Simulated):** Visual placeholders for Flash content (1996, 2000).
    *   **"Under Construction" Signs:** Classic animated GIFs.
    *   **ASCII Art Viewer:** Display of ASCII art (1985), sometimes clickable.
    *   **MySpace-like Profile Section & Media Player:** (2005).
*   **Immersive Transitions:**
    *   **Visual Transitions:** Static, glitch, or other effects when changing eras (respects "Reduce Motion" setting).
*   **UI/UX Style Guide Adherence:**
    *   **Fonts:** Era-specific fonts like VT323, Press Start 2P, Comic Neue.
    *   **Colors:** Palettes authentic to each period.
    *   **Effects:** CRT screen flickers, glitch transitions, scrolling marquees, blinking text (all respect "Reduce Motion").
*   **User Customization:**
    *   **MySpace Era (2005):** Users can customize the background and text colors of their "profile" section. Customizations are saved in local storage.
*   **State Persistence (Local Storage):**
    *   The last selected year is remembered between sessions.
    *   Guestbook entries are saved per era.
    *   MySpace profile color customizations are saved.
*   **Accessibility:**
    *   **Reduce Motion Toggle:** A footer option allows users to disable or significantly reduce animations and transitions site-wide. OS-level `prefers-reduced-motion` is also respected as a default.
    *   ARIA attributes used for interactive elements.
*   **Advanced Easter Eggs:**
    *   Beyond simple text popups, specific eras have unique hidden interactions:
        *   **2000:** Trigger a "Matrix" digital rain effect.
        *   **1996:** Experience a simulated "Rickroll."
        *   Other eras feature themed text-based surprises.
*   **Responsive Design:** While emulating old desktop experiences, the site aims for basic usability on modern devices.

## Tech Stack

*   **HTML5**
*   **CSS3** (including Tailwind CSS for utility classes and custom CSS for era-specific styling and animations)
*   **TypeScript**
*   **React 18**
*   **Google Fonts**
*   **@google/genai (Gemini API):** Powers the dynamic generation of era-specific news headlines and ad copy.

## How to Run

This is a static frontend application. To run it:

1.  Ensure all files (`index.html`, `index.tsx`, `constants.ts`, `types.ts`, `App.tsx`, `metadata.json`, and all files in the `components/` and `utils/` directories) are in the same root directory.
2.  Open `index.html` in a modern web browser that supports ES6 modules. JavaScript must be enabled.

No build step is currently required for this setup.
